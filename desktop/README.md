# Desktop (Python + Tkinter/CustomTkinter)

Десктопная версия проекта MIAS (Matching Intelligent Assistant System) - приложение для управления пользователями и проектами с возможностью AI-подбора кандидатов.

## Описание

Приложение предоставляет графический интерфейс для работы с системой MIAS. Включает аутентификацию, управление профилями, проектами и запуск AI-матчинга. Интегрируется с backend API.

## Функционал

- Аутентификация через Telegram
- Просмотр и редактирование профиля пользователя
- Создание и управление проектами
- Добавление участников в проекты
- Запуск AI-подбора кандидатов
- Просмотр результатов матчинга

## Технологии

- Python 3.12+
- CustomTkinter для GUI
- Requests для API
- uv для управления зависимостями

## Запуск

1. Установите uv: `pip install uv`
2. Синхронизируйте зависимости: `uv sync`
3. Активируйте виртуальное окружение: `uv run python main.py`
   Или: `source .venv/bin/activate` (Linux/Mac) / `.venv\Scripts\activate` (Windows), затем `python main.py`

## Структура проекта

- `main.py` - Точка входа приложения
- `pyproject.toml` - Конфигурация проекта и зависимостей
- `requirements.txt` - Список зависимостей (для совместимости)

## Настройки

Создайте `.env` файл в корне desktop/ с:

```
API_BASE_URL=http://localhost:8000
```

Замените на URL вашего backend.
